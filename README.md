# superstore-sales-dashboard
Interactive Sales Dashboard built using Power BI



📊 Superstore Sales Dashboard | Power BI Project

✨ An interactive Sales Analytics Dashboard built using Microsoft Power BI to analyze sales performance, profitability, and product trends across different regions, states, and product categories.

This dashboard helps stakeholders quickly identify top-performing products, profitable regions, and sales distribution patterns.


The objective of this dashboard is to:

📌 Analyze overall sales performance
📌 Identify top selling products
📌 Compare regional sales distribution
📌 Evaluate profitability across states
📌 Provide interactive business insights

🗂 Dataset Information

This project uses the Sample Superstore Dataset, a commonly used dataset for learning data analytics and visualization.

📋 Dataset Features
Column	Description
🗓 Order Date	Date when the order was placed
🌍 Region	Geographic sales region
🏙 City	Customer city
📦 Category	Product category
🧩 Sub-Category	Product sub classification
💰 Sales	Total sales amount
📈 Profit	Profit generated
🔢 Quantity	Number of items sold
📊 Dashboard Components
📌 KPI Metrics

At the top of the dashboard, key performance indicators provide a quick overview:

These KPIs help track the overall business performance.

📍 Visualizations
🏙 Sales of Each City

Visualization Type: Table

Displays total sales generated by each city.

✔ Helps identify high revenue cities
✔ Highlights low performing locations

📊 Sum of Sales by Sub-Category

Visualization Type: Column Chart

This chart compares sales across different product types:

📱 Phones
🪑 Chairs
📦 Storage
🗂 Binders
💻 Machines
🖨 Copiers
📎 Accessories

📌 Insight:
Phones and Chairs generate the highest sales among all sub-categories.

🍩 Sales Distribution by Sub-Category

Visualization Type: Donut Chart

Shows percentage contribution of each product category.

Example insights:

📱 Phones → ~14% of sales
🪑 Chairs → ~14% of sales
📦 Storage → ~9% of sales

This helps understand product demand distribution.

🌎 Sales by Region

Visualization Type: Bar Chart

Compares sales across different regions:

🌍 West
🌍 East
🌍 Central
🌍 South

📊 Key Insight

🥇 West region generates the highest revenue
📉 South region generates the lowest sales

💵 Profit by State

Visualization Type: Bar Chart

Displays profit generated by different states.

Top profitable states include:

🏆 California
🏆 New York
🏆 Washington

This helps identify regions contributing most to company profitability.

The dashboard includes interactive filters to explore data dynamically.

Users can filter data by:

☑ East

🧩 Sub-Category Filter

Users can select specific product categories such as:

📱 Phones
🪑 Chairs
📦 Storage
📎 Accessories

and many more...

This allows deep product level analysis.

📈 Key Business Insights

🔹 Phones and Chairs generate the highest sales revenue

🔹 The West region dominates total sales performance

🔹 California and New York generate significant profits

🔹 Some product categories like Art, Labels, and Fasteners contribute very little to sales
